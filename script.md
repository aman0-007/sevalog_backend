## Installed production dependencies
# express: Framework
# pg: PostgreSQL driver
# dotenv: Managing configurations safely
# cors: Cross-Origin Resource Sharing safety
# helmet: Secures HTTP headers to protect against common web vulnerabilities
# morgan: HTTP request logger for debugging

# -- npm install express pg dotenv cors helmet morgan bcryptjs jsonwebtoken --
